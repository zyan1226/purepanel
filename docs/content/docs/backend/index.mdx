---

使用 IDE 打开项目，你看到的目录结构应该如下：

```
├─ai -------------------------------------- AI相关
├─annotation ------------------------------ 注解
├─config ---------------------------------- 配置
├─constant -------------------------------- 常量
│  └─enums -------------------------------- 枚举
├─controller ------------------------------ 接口定义
├─exception ------------------------------- 异常相关
├─interceptor ----------------------------- 拦截器
├─log ------------------------------------- 日志相关
├─mapper ---------------------------------- MyBatis Mapper 接口
│  └─xml ---------------------------------- MyBatis XML 文件
├─pojo ------------------------------------ 实体类
│  ├─bo ----------------------------------- 业务对象
│  ├─entity ------------------------------- 数据库对应对象
│  ├─properties --------------------------- 配置对象
│  ├─req ---------------------------------- 请求对象
│  └─res ---------------------------------- 响应对象
├─service --------------------------------- 服务层
│  └─impl --------------------------------- 服务实现类
└─util ------------------------------------ 工具
```
在二次开发中，可按照该结构管理项目代码。
