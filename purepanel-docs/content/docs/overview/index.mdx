💧 PurePanel 是一个基于 [Spring Boot 3](https://spring.io/projects/spring-boot) 和 [Vue3]() 的**极致轻量的中后台项目快速启动脚手架**。

## 特性

1. **轻装上阵，开箱即用，** 没有太多复杂的预置功能，仅包含基本的用户、角色、权限体系。当然，在某些情况下，这可能显得些许“简陋”。
2. **现代化 UI 设计，** 基于 [TDesign](https://tdesign.tencent.com/) 构建。借鉴 macOS Tahoe 26。
3. **代码整洁，注释齐全，** 易读、易开发、易维护。
4. **集成 AI 助手（仍在探索中），** 最终目标是 AI 能够与业务高度融合，不只是简单的不懂业务的“聊天机器人”。

## 技术选型

1. **后端：** Spring Boot 3 + MyBatisPlus + MySQL + Redis + Swagger + Hutool
2. **前端：** Vue3 + TypeScript + TDesignVueNext + Vue Router + Pinia + Axios + Tailwindcss + Vite
